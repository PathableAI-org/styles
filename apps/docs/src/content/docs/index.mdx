Pathable Styles is the foundational SCSS/CSS package for PathableAI. It defines the core visual language — brand colors, typography, spacing, elevation, and semantic tokens — that all PathableAI products are built upon.

## What's included

This first version documents:

- **Style Foundations** — brand colors, semantic color roles, typography scale, spacing, elevation, and border-radius tokens
- **Usage Guidance** — how to install and consume `@pathable/styles` in your project
- **Agent-Facing Rules** — constraints and conventions for AI coding agents working with PathableAI styles

## What's not yet included

This is **not yet a complete component library**. Future versions will add:

- HTML/CSS example patterns
- React component examples
- Vue component examples
- A full component catalog
- Published npm package
