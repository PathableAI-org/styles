title: Foundations
description: The design tokens and style foundations that make up the Pathable visual language.
---

The Pathable design system is built on a set of foundational tokens organized into the following categories:

## Brand colors

Core brand palette for PathableAI products.

| Token                     | Role        |
| ------------------------- | ----------- |
| `--pathable-blue`         | Primary     |
| `--intelligent-jade`      | Secondary   |
| *(additional brand tokens)* |             |

## Semantic colors

Tokens that carry meaning and map to interface roles.

- **Text** — `--pathable-color-text`
- **Background** — `--pathable-color-bg`
- **Surface** — `--pathable-color-surface`
- **Accent** — `--pathable-color-accent`
- **Border** — `--pathable-color-border`

## Typography

| Token                     | Value  | Usage   |
| ------------------------- | ------ | ------- |
| `--pathable-font-heading` | Fredoka | Headings |
| `--pathable-font-body`    | Nunito  | Body text |

## Spacing scale

A consistent spacing scale using `--pathable-spacing-*` tokens (e.g., `--pathable-spacing-xs`, `--pathable-spacing-sm`, `--pathable-spacing-md`, `--pathable-spacing-lg`, `--pathable-spacing-xl`).

## Elevation

Box-shadow tokens for surface elevation (`--pathable-elevation-*`).

## Border radius

| Token                     | Value  |
| ------------------------- | ------ |
| `--pathable-radius-sm`    | Small  |
| `--pathable-radius-md`    | Medium |
| `--pathable-radius-lg`    | Large  |
| `--pathable-radius-full`  | Full   |
