title: Getting Started
description: How to install and use @pathable/styles in your project.
---

Add `@pathable/styles` to your project from GitHub Packages:

## Install

```bash
pnpm add github:PathableAI-org/styles
```

## Workspace usage

In a monorepo using pnpm workspaces, reference the package with the `workspace:*` protocol:

```json
{
  "dependencies": {
    "@pathable/styles": "workspace:*"
  }
}
```

## Import

Once installed, import the compiled stylesheet:

```css
@import '@pathable/styles/dist/styles.css';
```

This makes all design tokens — CSS custom properties — available in your project.

## Using tokens

Tokens are exposed as CSS custom properties. Use them directly in your stylesheets or component frameworks:

```css
.my-component {
  color: var(--pathable-color-text);
  background: var(--pathable-color-surface);
  font-family: var(--pathable-font-body);
  padding: var(--pathable-spacing-md);
  border-radius: var(--pathable-radius-md);
}
```
