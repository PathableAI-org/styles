Here's what's planned for future versions of Pathable Styles.

## HTML/CSS examples

Pre-built HTML/CSS patterns demonstrating common UI components styled with Pathable tokens.

## React examples

Ready-to-use React component examples that consume `@pathable/styles` tokens, including buttons, cards, forms, and layout primitives.

## Vue examples

Vue component examples mirroring the React patterns, providing parity across frameworks.

## Component catalog

A browsable catalog of all available components and patterns, integrated directly into this documentation site.

## npm publishing

Publish `@pathable/styles` to npm for simpler installs without GitHub Packages dependency.
